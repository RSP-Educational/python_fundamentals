# Python Fundamentals

Dieses Repository gibt einen Einstieg in die Python Programmierung.

`uebungen.ipynb` enthält theoretische Grundlagen und Übungsaufgaben.

> **Empfehlung: Nutzen Sie Google Colab**<br>
> - Kostenlose cloudbasiert online Plattform, die Code direkt im Browser schreiben und ausführen lässt: https://colab.research.google.com (Account benötigt)<br>
> - Vorteile:
> - Keine Installation nötig
> - Hardwareunabhängig
> - Keine Kompatibilitätsprobleme
> - Cloudbasiert ➡️ auf anderen Geräten über Benutzerzugang verfügbar

## How To

1. Gehen Sie auf https://colab.research.google.com
➡️ Datei  ➡️ Notebook öffnen ➡️ GitHub
